How to use the KiWELa keyboard on mobile

1. Dowload the tok_lentan 1.1.kmp file from here.
2. Intall "Keyman" on the appstore
3. Open the app
4. Click "add a keyboard for your language"
5. Click "install from local file"
6. Find and select the tok_lentan 1.1.kmp file
7. Click install and ok
8. Click the back arrow (<-)
9. Click "enable Keyman as a system wide keyboard"
10. Turn the Keyman slider on

This instruction set was made for Android, and may not be exactly the same on iPhone, but likely is. Feedback is appreciated.
